```

```

**開発環境**:
```
http://localhost:8888/
```

### ステップ2: ログイン

1. ブラウザでシステムURLを開く

- パスワード: 初回設定時に設定したもの

## 2. 基本操作（2分）

### 画面構成を理解する

```
┌──────────────────────────────────────────────────┐
│         ヘッダー（タイトル、接続状態）             │
├──────────┬───────────────────────────────────────┤
│ サイド   │            メインコンテンツ            │
│ メニュー  │         （各機能の画面）                │
│          │                                       │
│ • ダッシュボード                                   │
│ • ユーザー管理 (*)                                 │
│ • アプリ管理                                      │
│ • インシデント管理                                 │
│ • 変更管理                                        │
│ • 監査ログ (*)                                    │
│ • システム設定 (*)                                 │
└──────────┴───────────────────────────────────────┘

(*) = 管理者のみアクセス可能
```

### メニューを試す

1. 左側のサイドメニューから「インシデント管理」をクリック
2. インシデント一覧が表示される
3. 「ダッシュボード」をクリックして戻る

### 基本操作パターン

全ての機能で共通の操作パターン:

```
[新規作成]
  「新規〇〇」ボタン → フォーム入力 → 保存

[編集]
  「✏️」ボタン → フォーム編集 → 保存

[削除]
  「🗑️」ボタン → 確認ダイアログ → OK

[検索]
  検索ボックスに入力 → Enter
```

## 3. 主要機能を使ってみる（2分）

### 3-1. インシデント登録（30秒）

**インシデント = 既に発生している問題・障害**

1. サイドメニュー→「インシデント管理」
2. 「新規インシデント」ボタンをクリック
3. 以下を入力:
   - **タイトル**: 「テスト: 初めてのインシデント登録」
   - **対象アプリ**: 「経費精算」（または任意）
   - **優先度**: 「低」
   - **説明**: 「クイックスタートガイドの練習用データです」
4. 「保存」をクリック
5. 一覧に表示されることを確認

### 3-2. 変更要求作成（30秒）

**変更要求 = 新機能追加・改善の申請**

1. サイドメニュー→「変更管理」
2. 「新規変更要求」ボタンをクリック
3. 以下を入力:
   - **タイトル**: 「テスト: 初めての変更要求」
   - **対象アプリ**: 「営業管理」（または任意）
   - **タイプ**: 「改善」
   - **説明**: 「クイックスタートガイドの練習用データです」
4. 「保存」をクリック
5. 一覧に表示されることを確認

### 3-3. 検索・フィルタ（30秒）

1. インシデント一覧で検索ボックスに「テスト」と入力
2. 自分が作成したインシデントが表示される
3. 優先度フィルタで「低」を選択
4. フィルタされた結果が表示される

### 3-4. ログ確認（30秒）

1. サイドメニュー→「監査ログ」（管理者のみ）
2. 最近の操作ログが表示される
3. 自分の操作（インシデント作成等）が記録されている

## 5. よくある最初の質問

### Q: パスワードを忘れました
**A**: システム管理者に連絡してリセットを依頼してください。

### Q: 画面が表示されません
**A**: ブラウザのキャッシュをクリア（Ctrl+Shift+Delete）して再読み込みしてください。

### Q: データは自動保存されますか？
**A**: 「保存」ボタンをクリックしないと保存されません。ブラウザのlocalStorageに保存されます。

### Q: 他の人とデータを共有できますか？
**A**: 現在のバージョンでは、ブラウザごとに独立したデータです。バックアップ・リストア機能で共有できます。

### Q: スマートフォンでも使えますか？
**A**: はい、レスポンシブデザインに対応していますが、PC環境での利用を推奨します。

## 10. トラブルシューティング（1分診断）

### 問題: ログインできない
```
✅ ユーザー名・パスワードを確認
✅ 大文字・小文字を区別
✅ アカウントが有効か管理者に確認
```

### 問題: 画面が表示されない
```
✅ JavaScript有効化
✅ キャッシュクリア（Ctrl+Shift+Delete）
✅ 別のブラウザで試す
```

### 問題: データが保存されない
```
✅ 必須項目（赤い*）を全て入力
✅ localStorage容量を確認
✅ シークレットモードを使わない
```

### 問題: 画面が遅い
```
✅ ブラウザキャッシュクリア
✅ 古いログを削除
✅ ネットワーク速度確認
```
