# AppSuite 管理運用システム

DeskNet's Neo / AppSuite用の管理運用Webアプリケーションです。

## 機能一覧

### ダッシュボード
- ユーザー数・アプリ数の統計表示
- 未解決インシデント数・保留中の変更要求数表示
- 最近の操作ログ
- アプリサマリー

### ユーザー管理
- ユーザー一覧表示
- 検索・フィルタリング（ステータス、権限）
- ユーザー追加・編集・削除
- 権限管理（管理者/ユーザー）
- ステータス管理（有効/無効）

### アプリ管理
- AppSuiteアプリの一覧表示
- 検索・フィルタリング（カテゴリ、ステータス）
- アプリ追加・編集・削除
- レコード数表示
- ステータス管理（稼働中/メンテナンス/停止中）

### インシデント管理
- アプリのインシデント（不具合・障害）管理
- 検索・フィルタリング（優先度、ステータス）
- インシデント登録・編集・削除
- 優先度設定（高/中/低）
- ステータス管理（未対応/対応中/解決済/クローズ）
- 担当者割当

### 変更管理
- アプリの変更要求（機能追加・改善）管理
- 検索・フィルタリング（変更種別、ステータス）
- 変更要求登録・編集・削除
- 変更種別（新機能/改善/変更/バグ修正）
- ステータス管理（下書き/申請中/承認済/実施中/完了/却下）
- 実施予定日管理

### 監査ログ
- 全操作の記録・表示
- 日付範囲検索
- 操作タイプ・対象でのフィルタリング
- CSVエクスポート

### システム設定

#### API接続設定
- DeskNet's Neo API接続設定
- 認証方式選択（Bearer Token/Basic認証/APIキー）
- 接続テスト機能
- 自動同期間隔設定

#### 基本設定
- システム名・組織名設定
- 管理者メールアドレス設定
- テーマ選択（ライト/ダーク/システム）
- 言語選択（日本語/English）
- 日付形式設定
- 表示件数設定

#### 通知設定
- メール通知のオン/オフ設定
  - 新規インシデント登録時
  - 高優先度インシデント発生時
  - 変更要求の承認依頼時
  - 変更完了時
- SMTPサーバー設定
- テストメール送信機能

#### セキュリティ設定
- パスワードポリシー
  - 最小文字数
  - 大文字/数字/特殊文字必須設定
  - パスワード有効期限
- セッション設定
  - タイムアウト時間
  - 最大同時ログイン数
  - 二要素認証設定
- ログイン制限
  - ログイン失敗許容回数
  - アカウントロック時間

#### ワークフロー設定
- インシデント管理
  - 自動割当設定
  - デフォルト担当者
  - エスカレーション期限
- 変更管理
  - 承認必須設定
  - 承認者設定
  - リードタイム設定
- ステータス遷移ルール

#### バックアップ・リストア
- 自動バックアップ設定
- 手動バックアップ作成
- バックアップからのリストア
- データエクスポート
- データ初期化

## ファイル構成

```
itsm-admin/
├── index.html          # メインHTML
├── css/
│   └── styles.css      # スタイルシート
├── js/
│   ├── api.js          # API連携モジュール
│   ├── modules.js      # 機能モジュール（ユーザー、アプリ、ログ）
│   └── app.js          # メインアプリケーション
└── README.md           # このファイル
```

## 使用方法

1. `index.html`をブラウザで開く
2. 「API設定」でDeskNet's NeoのAPI情報を設定（オプション）
3. 各機能を使用

## 技術仕様

- **フレームワーク**: Vanilla JavaScript
- **スタイリング**: カスタムCSS
- **アイコン**: Font Awesome 6
- **外部依存**: なし

## ブラウザ対応

- Chrome（推奨）
- Firefox
- Safari
- Edge

## API連携

DeskNet's Neo V7.0以降のAPIに対応しています。
APIキーは管理画面から取得してください。
