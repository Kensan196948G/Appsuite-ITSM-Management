## 1. プロジェクト概要

### 1.1 プロジェクトの目的
DeskNet's Neo AppSuite上で動作するITSM（ITサービス管理）システムを開発し、ユーザー管理、アプリ管理、インシデント管理、変更管理、監査ログ、システム設定の各機能を提供する。

### 1.2 開発期間
| 項目 | 期間 |
|------|------|
| 全体期間 | 約6.5ヶ月（26週間） |
| 開始予定 | 2026年2月1日 |
| 完了予定 | 2026年8月15日 |

### 1.2.1 期間延長の理由
**Phase 0-Extended（環境構築強化）を追加**:
- 全SubAgent機能（7体構成）の設定
- Git Worktree構成とHooks機能の実装
- 開発/本番環境の完全分離
- 並列開発基盤の整備

### 1.3 開発体制
| 役割 | 人数 | 担当内容 |
|------|------|----------|
| プロジェクトマネージャー | 1名 | 全体管理・進捗管理・リスク管理 |
| システムアーキテクト | 1名 | 技術設計・アーキテクチャ決定 |
| フロントエンド開発者 | 2名 | UI/UX実装・JavaScript開発 |
| バックエンド開発者 | 2名 | API連携・データ処理 |
| QAエンジニア | 1名 | テスト設計・実行・品質管理 |
| テクニカルライター | 1名 | ドキュメント作成・保守 |

### 3A.3 SubAgent構成（7体）

| # | Agent名 | 専門領域 | ツール | モデル |
|---|---------|----------|--------|--------|
| 1 | Bash Agent | Git/CLI操作 | Bash, Read, Write | haiku |
| 2 | General Purpose | 汎用タスク | All tools | sonnet |
| 3 | Statusline Setup | UI設定 | Read, Edit | haiku |
| 4 | Explore Agent | コード探索 | Glob, Grep, Read | haiku |
| 5 | Plan Agent | 設計・計画 | All except Edit/Write | sonnet |
| 6 | Claude Code Guide | ガイド・ヘルプ | Read, WebFetch | haiku |
| 7 | Code Simplifier | リファクタリング | Read, Edit, Grep | sonnet |

### 3A.4 Hooks機能

| Hook種別 | タイミング | 処理内容 |
|----------|-----------|----------|
| UserPromptSubmit | ユーザー入力時 | タスク開始ログ |
| PreToolUse | Edit/Write前 | Git status確認 |
| PostToolUse | Edit/Write後 | Lint/Test実行 |
| PostToolUseFailure | ツール失敗時 | エラーログ記録 |
