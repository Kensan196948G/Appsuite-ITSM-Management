# WebUI Sample アクセシビリティ手動検証チェックシート

更新日: 2026-02-24
対象: `WebUI-Sample/index.html`

## 1. フォーカス順（キーボード操作）

- [ ] 初期表示時、`Tab` で上部ヘッダ操作（ロール切替 / バックアップ / 新規登録）へ移動できる
- [ ] サイドバー各ナビ項目へ `Tab` で到達できる
- [ ] 設定画面の各タブ（API/基本/通知/セキュリティ/ワークフロー/権限/バックアップ）へ順に移動できる
- [ ] フォーカスが見える（ブラウザ標準またはカスタム輪郭）

## 2. モバイルナビ切替（レスポンシブ）

- [ ] 画面幅 1080px 以下で `#menuToggle` が表示される
- [ ] `#menuToggle` 押下でサイドバーが開閉する
- [ ] `#menuToggle` の `aria-expanded` が `false -> true -> false` に切り替わる
- [ ] サイドバー外クリックで閉じる
- [ ] `Esc` キーで閉じる
- [ ] 閉じた後にフォーカスが `#menuToggle` に戻る

## 3. モーダル操作（作成/編集）

- [ ] 任意の「新規」ボタンでモーダルが開く
- [ ] モーダル表示直後にモーダル内の最初の操作要素へフォーカスが移る
- [ ] `Tab` / `Shift+Tab` でフォーカスがモーダル外に抜けない（フォーカストラップ）
- [ ] `Esc` でモーダルが閉じる
- [ ] モーダルを閉じた後、元のボタンへフォーカスが戻る

## 4. スクリーンリーダー観点（最低限）

- [ ] `#menuToggle` に `aria-label` がある
- [ ] モーダルに `role="dialog"` `aria-modal="true"` `aria-labelledby` がある
- [ ] トースト領域に `aria-live="polite"` がある
- [ ] サイドバーにランドマーク/ラベル（`aria-label`）がある

## 5. 権限制御のUI反映

- [ ] 一般ユーザー切替で `設定` ナビが無効表示になる
- [ ] 一般ユーザー切替で `監査ログCSVエクスポート` が disabled になる
- [ ] 一般ユーザー切替で権限設定・Docs設定の保存ボタンが disabled になる
